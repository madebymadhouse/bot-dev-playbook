# Runbooks

Step-by-step guides for repeated or risky bot-development tasks.

## Current Runbooks

| Runbook | When to use it |
|---|---|
| [local-setup.md](./local-setup.md) | First-time setup on a new machine |
| [command-deployment.md](./command-deployment.md) | Deploying slash commands to guild or global |
| [docker-deploy-and-verify.md](./docker-deploy-and-verify.md) | Deploying a stack update via Docker Compose |
| [incident-triage.md](./incident-triage.md) | Bot is down or behaving unexpectedly |
| [create-new-playbook.md](./create-new-playbook.md) | Turn repeated chat guidance into a durable repo |

## Format

Keep each runbook short and in this order:

1. Purpose
2. Preconditions
3. Exact steps with copy-paste commands
4. Verification
5. Failure handling
6. What docs to update afterward
